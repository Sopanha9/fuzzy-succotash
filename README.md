## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+ recommended)

### Installation

```bash
# Clone the repo
git clone https://github.com/Sopanha9/fuzzy-succotash.git

# Navigate into the project
cd fuzzy-succotash

# Install dependencies
npm install
```

### Run Locally

```bash
npm start
```

Then open your browser and go to `http://localhost:3000`
